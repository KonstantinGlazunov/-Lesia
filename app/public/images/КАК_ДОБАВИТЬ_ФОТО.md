# Фотографии с Instagram для сайта

Скачайте нужные фото из [профиля lesia.psy5748](https://www.instagram.com/lesia.psy5748/) и сохраните в эту папку под указанными именами.

## Куда класть файлы

### В папку `public/images/` (эта папка)

| Имя файла        | Где используется      |
|------------------|----------------------|
| `hero-bg.jpg`     | Фон главного экрана (Hero) |
| `master-photo.jpg` | Фото мастера в блоке «О мастере» (About) |

### В папку `public/images/portfolio/`

Фото работ для портфолио (9 штук). Имена файлов:

- `brows-1.jpg`, `brows-2.jpg`, `brows-3.jpg` — работы по бровям  
- `lips-1.jpg`, `lips-2.jpg`, `lips-3.jpg` — работы по губам  
- `eyes-1.jpg`, `eyes-2.jpg`, `eyes-3.jpg` — работы по векам  

Можно заменить на свои имена: тогда нужно будет поправить пути в `src/sections/Portfolio.tsx` и, при необходимости, в `About.tsx` и `Hero.tsx`.

## Как скачать фото из Instagram

1. Откройте [instagram.com/lesia.psy5748](https://www.instagram.com/lesia.psy5748/).
2. Выберите публикацию → три точки «⋯» → «Сохранить» или скопировать ссылку.
3. Сохранить изображение на компьютер можно через приложение или сайты-загрузчики (по правилам Instagram и авторским правам).
4. Переименуйте файлы по таблице выше и положите в `app/public/images/` и `app/public/images/portfolio/`.

После добавления файлов перезапустите `npm run dev` и обновите страницу — фото подхватятся автоматически.
